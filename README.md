# Practice-Challenge-2

Write a function that accepts a string and returns an Object with two properties(an array of most frequent characters, and their count).

For eg. 

```
"test" =>{
          char:['t'],
          count:2
          }
```
```
"string"=>{
           char:['s','t','r','i','n','g'],
           count:1
           }
```
```
"giraffe"=>{
            char:['f'],
            count:2
            }
  ```
  ```
"aeroplane"=>{
              char:['a','e'],
              count:2
              }
```
